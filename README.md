## Getting Started
To get a local copy up and running follow these simple example steps.

- Fork and then Clone this repo using this URL (https://github.com/tashisrael/React-Bookstore.git) 
- Or download it as a ZIP format from the green button above with the text "Code".

### Install
```
npm install
```
### Usage
To experience a Demo on your local machine run:
```
npm start
```
